# 保存帖 - iPhoneアプリ化の手順

このフォルダの中身をGitHub Pagesにアップロードすると、
iPhoneの「ホーム画面に追加」でアプリのように使えるようになります。

## フォルダの中身
- index.html … アプリ本体
- data.js … 食品データ
- manifest.json … PWA設定ファイル
- sw.js … オフライン対応用スクリプト
- apple-touch-icon.png / icon-192.png / icon-512.png … アイコン
- favicon-16.png / favicon-32.png … ブラウザ用の小さいアイコン

**この9個のファイルは、すべて同じフォルダの直下に置いてください。**
（サブフォルダに分けると読み込めなくなります）

---

## 手順1：GitHubアカウントを作る（すでにあれば飛ばしてOK）
1. https://github.com/ にアクセス
2. 「Sign up」からメールアドレスで無料登録

## 手順2：リポジトリ（保存場所）を作る
1. 右上の「+」→「New repository」
2. Repository name に好きな名前を入力（例：`hozoncho`）
3. 「Public」を選択（Privateだと無料プランではPages機能が使えません）
4. 「Create repository」をクリック

## 手順3：ファイルをアップロード
1. 作成したリポジトリの画面で「uploading an existing file」をクリック
   （もしくは「Add file」→「Upload files」）
2. このフォルダの中の9個のファイルをすべてドラッグ＆ドロップ
3. 下にスクロールして「Commit changes」をクリック

## 手順4：GitHub Pagesを有効にする
1. リポジトリ画面上部の「Settings」タブをクリック
2. 左メニューの「Pages」をクリック
3. 「Branch」を `main`、フォルダを `/ (root)` にして「Save」
4. 数分待つと、ページ上部に
   `https://（あなたのユーザー名）.github.io/（リポジトリ名）/`
   というURLが表示されます

## 手順5：iPhoneでホーム画面に追加
1. iPhoneのSafari（Chromeではなく**Safari**である必要があります）で、
   手順4のURLを開く
2. 画面下の「共有」ボタン（□に↑の矢印のアイコン）をタップ
3. 「ホーム画面に追加」をタップ
4. 名前を確認して「追加」

これでホーム画面にアイコンが追加され、タップするとブラウザのバー等が
表示されないアプリのような画面で開きます。

---

## 注意点
- **必ずSafariから追加してください。** Chromeなど他のブラウザからは
  「ホーム画面に追加」してもこの機能（フルスクリーン表示）が使えません。
- 内容を更新したいときは、GitHubのリポジトリ画面でファイルを
  再アップロード（上書き）すれば、次にアプリを開いたときに反映されます。
- 完全無料で、期限もありません。
